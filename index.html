Star Wars jQuery Game

The Star Wars jQuery game was project that was developed along a strict set of rules. The player chooses a character, and after each attack the players attack is doubled. There is a win condition for every character. This site features a 'Star Wars' themed design, complete with custom logos, and an introductory video poking fun at our class's development pace. Technologies employed include HTML, CSS, vanilla Javascript, as well as jQuery.

Heavy Metal Quiz Game

Do you like heavy metal? Do you like quiz games? Well then, this website is for you! The website tracks keypress events and allows the user to play a hangman style game. Upon completion, the user is shown a photo, as well as an audio sample from the artist in question. This website features a 'metal' themed logo and asset set, either by my own design or drawn from material in the public domain. The fonts match the asthetic as well. The technologies employed include HTML, CSS, vanilla Javascript as well as jQuery.

Harry Potter Giftastic

There is a new course being offered at Hogwarts... Magical GIFs! This site utilizes REST API's in order to provide the user a unique experience... When the user "prepares" a spell, they can then use it to display an assortment of GIF's from Giphy's collection of GIFs on that subject. The user can also "sort" themselves using infomation drawn for the Harry Potter API. This site features a Harry Potter theme, with logos of my own design, as well as a series of font faces that fit the overall design goal. The technologies utilized include HTML, CSS, vanilla Javascript, jQuery, and REST APIs.

Train Scheduler

Hop aboard Magical Railways to reach your ultimate destination! This assignment utilizes a cloud based database, Google Firestore, in order to provide the user a fictional train schedule. The next train arrival time is calculated with MomentJS, a Javscript library that allows for the easy manipulation of datetime objects. Each trainstop relates to a location in the children's game Candyland, but feel free to add your own! Once a train is added, the database will update in real time and the information will be accessible by ever visitor to the site. The technologies employed include HTML, CSS, Javascript, jQuery, MomentJS, and Google Firebase.
